Requirements:
Norbyte's Baldur's Gate 3 Script Extender 2.0+ (https://www.nexusmods.com/baldursgate3/mods/2172)

Compatibility: 
Compatible with everything.

Installation:
Drop .pak from archive to user/appdata/local/larian studios/baldur's gate 3/mods or install with BG3 Mod Manager (suggested mod manager for BG3)/Vortex. In Mod Manager mod should be in Overrides. Mark mod as replacer in Vortex.

Uninstallation:
Remove mod's .pak manually or with BG3 Mod Manager/Vortex.
